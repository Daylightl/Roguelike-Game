# 鸠摩智转刀 - Roguelike Game

一个基于纯前端技术开发的轻量化Roguelike生存游戏，灵感来自吸血鬼幸存者，具有搞怪武侠风格和解压易上手的游戏体验。

## 游戏特色

- 🎮 **纯前端实现** - 无需后端支持，双击HTML即可开始游戏
- ⚔️ **自动旋转攻击** - 飞刀围绕主角自动旋转，轻松清场
- 🎯 **走位躲避** - 灵活的WASD/方向键/鼠标控制
- 📈 **升级成长** - 9种技能自由搭配,打造专属流派
- 👹 **多样敌人** - 小兵、精英、BOSS三种敌人类型
- 🎁 **宝箱奖励** - 随机掉落强力增益效果
- 🏆 **BOSS战** - 击败BOSS取得最终胜利

## 快速开始

### 方式一：使用本地服务器（推荐）⭐

为了确保音频和图片正常加载，建议使用本地HTTP服务器运行游戏：

**Python 3:**
```bash
cd Roguelike-Game
python -m http.server 8000
```

**Node.js:**
```bash
cd Roguelike-Game
npx http-server -p 8000
```

**VS Code:**
安装 "Live Server" 插件，右键 `index.html` 选择 "Open with Live Server"

然后在浏览器访问：`http://localhost:8000`

### 方式二：直接打开（可能有限制）

直接双击 `index.html` 文件也可以运行，但可能会遇到：
- ⚠️ 音频无法播放（浏览器跨域限制）
- ⚠️ 部分功能受限

### 推荐浏览器

- Chrome 90+
- Edge 90+
- Firefox 88+

## 操作说明

### PC端控制

**键盘操作：**
- `WASD` 或 `方向键` - 控制主角移动
- `空格键` - 强制吸收周围经验球

**鼠标操作：**
- `点击` - 主角自动向鼠标位置移动
- `拖动` - 主角跟随鼠标持续移动

## 游戏玩法

### 核心循环

1. **自动攻击** - 飞刀自动旋转，消灭周围敌人
2. **拾取经验** - 击杀敌人掉落经验球
3. **升级选技** - 经验满100升级，从3个技能中选择1个
4. **击败BOSS** - 等级达到10或存活120秒后，BOSS出现

### 角色属性

| 属性 | 初始值 | 说明 |
|------|--------|------|
| 生命值 | 100 | 降至0游戏失败 |
| 移动速度 | 100 px/s | 通过技能提升 |
| 飞刀数量 | 2 | 最多8把 |
| 飞刀伤害 | 10 | 可通过技能提升 |
| 旋转半径 | 80 px | 最大150px |

### 敌人类型

| 类型 | 生命值 | 移速 | 伤害 | 经验值 | 特殊能力 |
|------|--------|------|--------|--------|----------|
| 小兵 | 20 | 80 | 5 | 5 | 无 |
| 精英 | 80 | 100 | 10 | 20 | 减速玩家 |
| BOSS | 300 | 120 | 15 | 100 | 范围冲击 |

### 技能系统

游戏提供9种技能，分为三大类：

**进攻类：**
- 飞刀+1 - 增加1把旋转飞刀
- 飞刀伤害+10% - 永久提升伤害
- 旋转半径+10px - 扩大攻击范围
- 加速转刀 - 提升25%旋转速度

**生存类：**
- 生命值+20 - 提升生命上限并回复
- 移动速度+5% - 提升移速
- 防御强化 - 减少10%受到的伤害
- 紧急救援 - 立即回复40点生命

**功能类：**
- 经验磁铁 - 瞬间吸收全场经验球

### 宝箱系统

宝箱每45秒刷新一次，触碰开启后随机获得以下奖励之一：

- **刀刃补给** - 获得4把额外飞刀
- **锋利淬炼** - 武器伤害永久+10
- **自动拾取** - 30秒内经验球自动吸附
- **移速爆发** - 移速提升50%持续10秒
- **武器库** - 解锁随机武器外观

### 胜利条件

**击败BOSS** - 当玩家等级≥10或存活时间≥120秒时，BOSS出现，击败BOSS即可胜利！

### 失败条件

**生命值归零** - 主角生命值降至0时游戏结束

## 得分系统

```
最终得分 = 击杀数 × 10 + 存活秒数 × 2 + BOSS击杀奖励 × 50
```

## 开发技术

### 核心技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式与动画
- **JavaScript (ES6+)** - 游戏逻辑
- **Canvas 2D** - 图形渲染

### 架构设计

```
game.js
├── Utils - 工具函数库
├── AudioManager - 音效管理器
├── Player - 玩家类
├── Enemy - 敌人类（小兵/精英/BOSS）
├── ExpOrb - 经验球类
├── Chest - 宝箱类
├── Particle - 粒子效果
└── Game - 游戏主类（主循环/渲染/UI更新）
```

### 性能优化

- 使用对象池减少GC压力
- 圆形碰撞检测优化
- 帧率限制防止性能波动
- 粒子效果生命周期管理

## 音效支持（可选）

游戏已预留音效接口，如需完整音效体验，请准备以下音频文件并放置在 `assets/audio/` 目录：

- `bgm.mp3` - 背景音乐
- `hit.mp3` - 攻击命中音效
- `levelup.mp3` - 升级/拾取经验音效
- `chest_open.mp3` - 宝箱开启音效
- `boss_appear.mp3` - BOSS登场音效
- `gameover.mp3` - 游戏结束音效

注：无音效文件时游戏仍可正常运行。
